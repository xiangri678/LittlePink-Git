# LittlePink Project

This project, **LittlePink**, is a practice application inspired by Xiaohongshu, created during my self-learning journey with UIKit. To gain hands-on experience with UIKit and replicate features of Xiaohongshu.

### Project Details:
- **Development Environment**:  
  - **Xcode 16**  
  - **macOS 15**  
  - **M1 Chip**  
- **Testing**:  
  - Conducted on the **iPhone 16 Pro simulator**.    

I’ve also compiled some personal learning notes, which are available on the [following webpage](https://fluttering-raja-b46.notion.site/UIKit-156e97fc64be807dbc03c72ab60c58c4?pvs=4) and will be updated automatically based on my study progress.

### Current Progress:
1. **Static Setup of the Discovery Page**  
2. **Top Sliding Tab Bar**  
   - Implemented using the `XLPagerTabStrip` library.  
3. **Waterfall Layout for Content Display**  
   - Built with the `CHTCollectionViewWaterfallLayout` library.  
4. **Icons and Splash Screen**  
5. **Bottom Navigation Bar**  
   - Added icons for all five tabs.

# 小粉书项目

本项目为一个模仿小红书的练习应用，命名为 **小粉书**，是我在自学 UIKit 时完成的实战项目。通过模仿小红书的部分功能，熟悉 UIKit 开发流程并积累实战经验。

### 项目详情：
- **开发环境**：  
  - **Xcode 16**  
  - **macOS 15**  
  - **M1 芯片**  
- **测试环境**：  
  - 在 **iPhone 16 Pro 模拟器**上进行测试。  

我将部分学习笔记整理并发布在[以下网页](https://fluttering-raja-b46.notion.site/UIKit-156e97fc64be807dbc03c72ab60c58c4?pvs=4)中，根据学习进度，不定期自动更新。

### 当前进度：
1. **静态搭建发现页**  
2. **顶部滑动切换栏**  
   - 使用 `XLPagerTabStrip` 库实现。  
3. **瀑布流页面**  
   - 使用 `CHTCollectionViewWaterfallLayout` 库实现。  
4. **图标与开屏页面**

   - 添加了五个导航栏的图标。
